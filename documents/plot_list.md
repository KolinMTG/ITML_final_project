# Plot List

Use this file to describe all registered .png in the plots folder. (Only in english please)

- [knn_neighbors_metrics](../plots/knn_neighbors_metrics.png) : KNN performance for different numbers of neighbors, evaluated using accuracy, precision, recall, and F1-score, for classifying wine into two categories (white vs. red).

- [regression_multiple_model_perf](../plots/regression_multiple_model_perf.png) : Relative performance of multiple models, normalized to the top-performing ExtraTrees model, for wine-quality classification. Added real performances for top3 best models. Perf = Error

- [alcool_kde](../plots/alcool_kde.png) : Kernel density plot showing the distribution of alcohol content across wines.Highlights where most samples concentrate and how spread or skewed the values are.

- [free_vs_total_sulfure_dioxide_kde](../plots/free_vs_total_sulfure_dioxide_kde.png) :   KDE comparison between free and total sulfur dioxide. Medians and quartiles help visualize differences in concentration ranges and variability.

  Helps observe the typical acidity range and how tightly values are clustered.

  Makes it easier to see dominant quality levels and possible class imbalance.
